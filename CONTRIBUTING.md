# Contributing

Thank you for wanting to contribute! Please follow these steps:

1. Read the `README.md` to understand the project.
2. Open an issue to discuss large changes before implementing them.
3. Fork the repository and create a branch with a clear name: `feature/<name>` or `fix/<issue>`.
4. Add tests where applicable and make sure `pytest` passes.
5. Format code with `black` and fix lint issues with `flake8`.
6. Keep commits small and descriptive.
7. Open a Pull Request describing the change and reference related issues.

Thank you for your contribution!
